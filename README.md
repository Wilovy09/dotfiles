# Setup

Este es mi intento de dotfiles.

- [Helix-Editor](https://helix-editor.com/) es mi editor principal.
- [VSCode](https://code.visualstudio.com/) es mi editor secundario.

- [Superfile](https://superfile.netlify.app/) | [xplr](https://xplr.dev/) son mis exploradores de archivos principales.
- [Thunar](https://docs.xfce.org/xfce/thunar/the-file-manager-window) es mi editor secundario.

- [Arch](https://aur.archlinux.org/) btw.
- [Hyprland](https://hyprland.org/) es mi escritorio.
- [Waybar](https://github.com/Alexays/Waybar) es mi barra de status.
- [GoLS](https://github.com/Wilovy09/GoLS) es mi LS.
- [Kitty](https://sw.kovidgoyal.net/kitty/) es mi terminal.
- `ZSH` es mi shell.
- [Oh-my-zsh](https://ohmyz.sh/) para perzonalizar.

- [JetBrainsNerd Mono](https://www.jetbrains.com/lp/mono/) es mi fuente.
- [LazyGit](https://github.com/jesseduffield/lazygit) es lo que uso para trabajar con git a parte del propio git.
