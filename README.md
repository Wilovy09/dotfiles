# Setup

Este es mi intento de dotfiles.

- `Helix-Editor` es mi editor principal.
- `VSCode` es mi editor secundario.

- `Superfile` | `xplr` son mis exploradores de archivos principales.
- `Thunar` es mi editor secundario.

- `Arch` btw.
- `Hyprland` es mi escritorio.
- `Waybar` es mi barra de status.
- `GoLS` es mi LS.
- `Kitty` es mi terminal.
- `ZSH` es mi shell.
- `Oh-my-zsh` para perzonalizar.

- `JetBrainsNerd Mono` es mi fuente.
- `LazyGit` es lo que uso para trabajar con git a parte del propio git.
